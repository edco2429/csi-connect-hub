
# CSI_Registration

CSI_Registration is a web application for managing Computer Society of India (CSI) events, registrations, and attendances. It provides separate dashboards for students, teachers, and committee members with role-specific functionalities.

## Features

### Student Dashboard:
- View upcoming events
- Register for events
- Track registration status
- Process event fee payments
- Manage user profile
- Personal calendar
- Notification system
- Settings dashboard

### Teacher Dashboard:
- Manage student registrations
- Monitor payments
- Attendance management
- Calendar management
- Notification system
- Settings dashboard

### Committee Dashboard:
- Event management
- Registration tracking
- Committee members section
- Attendance history
- Calendar management
- Notification system
- Settings dashboard

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Supabase (for backend)
- Vite
